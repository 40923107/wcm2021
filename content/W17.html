Gitlab 同步

同步影片：https://youtu.be/6OatJjjBB84

Gitlab：https://gitlab.com/40923107/wcm2021

已先用 Github 帳戶登入 Gitlab

1.到 puttygen Load 之前設的 SSH key

2.複製 Key 貼到 Gitlab 後，點 Add key

3.開啟 putty.exe ， Save "gitlab.com"，之後確認 proxy ，在確認 Auth 裡的選取檔案是否正確，最後點 Open

4.之後會跳出詢問視窗，點"是"

5.開啟 .git 中的 config 編輯畫紅線處，最後 git push gitlab
