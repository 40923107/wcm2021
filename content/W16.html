1.搜尋 Fossil scm，到 Download 下載 fossil.exe，看電腦支援 32 位元或 64 位元，下載到 Y 槽

2.開啟 ipv4.bat

3. fossil open 後，複製 wcm2021裡面的檔案，貼到40923107的資料夾

4.之後若要同步 Github 的倉儲內容，需要先 git pull ，再 fossil commit -m "initial add"。
